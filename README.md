## 🛠 Tecnologías Utilizadas

Las tecnologías utilizadas en el desarrollo del proyecto son:

- **Lenguaje de programación:** JavaScript (Node.js)
- **Entorno de ejecución:** Node.js v24.13.0
- **Framework backend:** Express
- **ORM:** Sequelize
- **Base de datos:** PostgreSQL
- **Autenticación:** JSON Web Tokens (jsonwebtoken)
- **Encriptación:** bcryptjs
- **Validación de datos:** express-validator
- **Manejo de errores:** http-errors
- **Logs:** Pino y pino-pretty
- **Configuración de entorno:** dotenv
- **CORS:** cors
- **CLI de base de datos:** sequelize-cli
- **Control de versiones:** Git

Las tecnologías fueron seleccionadas considerando criterios de rendimiento, escalabilidad, seguridad y su amplia adopción en entornos profesionales y académicos.
